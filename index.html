Javascript and JQuery Learning

Flashcards

Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

String

The String() function converts the value of an object to a string.

Modulo

The modular operator (%) returns the division remainder.

Operators

The definition of an operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

NaN

The NaN property represents "Not-a-Number" value. This property indicates that a value is not a legal number.

Variable Naming Conventions

JavaScript variables are containers for storing data values.

+=

+ means to add, and = is equals.

Function

A JavaScript function is a block of code designed to perform a particular task.

Method

JavaScript methods are the actions that can be performed on objects.

Argument to a method

Function arguments are the real values passed to (and received by) the function.

Chaining Methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Escape

This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

Alert

The alert() method displays an alert box with a specified message and an OK button.

Commenting Code

JavaScript comments can be used to explain JavaScript code, and to make it more readable.

Booleans

A JavaScript Boolean represents one of two values: true or false.

Undefined

The undefined property indicates that a variable has not been assigned a value.

Parameter

A Parameter is a variable that is passed to a function as part of the function call. The parameters that you pass supply values to the arguments defined within the function.

Return

The return statement ends function execution and specifies a value to be returned to the function caller.
